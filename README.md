# Soluções para Exercícios de Lógica e Programação

Este repositório contém a resolução de cinco exercícios de lógica e programação, utilizando a linguagem Python.

## Índice
1. [Cálculo do valor da variável `SOMA`](#1-cálculo-do-valor-da-variável-soma)

## 1. Cálculo do valor da variável `SOMA`

O código realiza a soma dos números inteiros de 1 até 13 utilizando um loop. Ao final do processo, o valor total da soma é armazenado em uma variável.

---

## 2. Verificação de número na sequência de Fibonacci

Este programa verifica se um número informado pelo usuário pertence à sequência de Fibonacci. Ele calcula a sequência até alcançar ou ultrapassar o número dado e retorna o resultado da verificação.

---

## 3. Análise de faturamento diário de uma distribuidora

O programa lê os dados de faturamento diário de uma distribuidora e retorna o menor e o maior valor de faturamento. Além disso, calcula quantos dias o faturamento foi superior à média mensal, ignorando dias sem faturamento.

---

## 4. Cálculo do percentual de faturamento por estado

Este programa calcula o percentual de participação no faturamento total de cada estado de uma distribuidora, com base nos valores de faturamento mensal fornecidos.

---

## 5. Inversão de string sem usar funções prontas

O programa inverte os caracteres de uma string fornecida pelo usuário sem utilizar funções prontas de inversão. A inversão é feita manualmente por meio de um loop que constrói uma nova string com os caracteres na ordem inversa.

---

## Requisitos

- Python 3.8 ou superior

## Como executar

1. Clone o repositório.
2. Navegue até o diretório do projeto.
3. Execute os programas diretamente no terminal ou em um ambiente de desenvolvimento Python.
